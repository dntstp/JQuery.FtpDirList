# JQuery.FtpDirList
Sipmle plugin to access FTP folder via PHP
##Requirements
    "jquery": "~1.12.0",
    "bootstrap": "^3.3.6"
##Usage
    paste ftp.php in root of your project
    edit ftp.php to set up your connection params
    attach JQuery.FtpDirList.js after loading jquery
    $('your-elem').FtpDirList();
